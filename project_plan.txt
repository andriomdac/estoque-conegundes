O plano é o seguinte:

Desenvolver uma api que armazena produtos, criando um estoque para a loja.
As informações do produto: 

Produto = (
    Nome,
    Descrição Geral,
    Categoria,
    Marca, 
    Código de Barras.
)

Essa vai ser a instância geral Produto (Isto é, os dados do produto que não mudam),
que vai possuir um código de barras único para
cada produto. O usuário, ao cadastrar um produto, vai ter uma verificação:
Se a instância de tal produto já existir no banco de dados, vai ser utilizada
a instância existente. Se não, será criado uma nova instância do novo produto.

Assim, ao longo do tempo, haverá cada vez menos necessidade de cadastro dos produtos
manualmente, vai ter apenas a busca na api e atribuir esse produto à loja do cliente.


Agora, após ser criada a instância geral (Produto), é necessário criar a instância
particular (Item_Produto), que são dados a respeito deste produto que diferem
de loja pra loja, ou podem ser alterados ao longo do tempo.

Item_Produto = (
    quantidade em estoque,
    descrição particular (
        cada loja pode descrever seu produto da forma que quiser,
        caso contrário, será utilizada a descrição geral
        ),

    dados da instância = (
        data da criação desta instância,
        produto = Instância Geral do Produto,
        loja = Instância da Loja,
    )
)


Item_Produto_Preço = (
    preço de custo,
    preço de venda,
    lucro,
    data de criação,
)
